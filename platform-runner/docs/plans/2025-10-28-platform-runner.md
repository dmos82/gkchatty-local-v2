# Platform Runner - Implementation Plan

## Project Overview

**Project Name:** Platform Runner
**Tech Stack:** Next.js 15 + Phaser 3 + TypeScript
**Game Type:** 2D side-scrolling platformer (Mario-style)
**Target Viewport:** 800x600 pixels
**Estimated Total Time:** 24 hours

## Architecture Summary

### Frontend Structure
- **Framework:** Next.js 15.0.3 with App Router
- **Game Engine:** Phaser 3.80+ with Arcade Physics
- **UI:** React 19 server components + Phaser client component
- **Asset Generation:** PixelLab MCP for sprite generation

### Phaser Scene Architecture
1. **BootScene** - Asset preloading
2. **MenuScene** - Main menu and start screen
3. **GameScene** - Core gameplay loop
4. **GameOverScene** - Game over screen with restart
5. **VictoryScene** - Victory screen after level 3

### Entity Classes
- **Player** - 4-directional sprites (idle, walk-left, walk-right, jump)
- **Goomba** - Ground patrol enemy with reversing AI
- **FlyingEnemy** - Sine wave flying enemy
- **Coin** - Collectible with score increment

### Manager Classes
- **LevelManager** - Load and spawn level data (platforms, enemies, coins)
- **ScoreManager** - Track score, lives, level progress
- **AssetManager** - (Handled by BootScene for this project)
